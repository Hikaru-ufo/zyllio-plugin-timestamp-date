# Zyllio Plugin - Timestamp Date Lisible

Plugin Zyllio pour afficher des dates au format lisible `JJ/MM/AAAA HH:MM:SS` (heure locale du device).

## Fonctions disponibles

### `Date heure maintenant`
Retourne la date et l'heure actuelle au format `JJ/MM/AAAA HH:MM:SS`.

- **Aucun paramètre requis**
- **Retourne** : texte (ex: `25/02/2026 14:35:22`)

### `Date heure depuis une date`
Retourne une date Zyllio formatée en `JJ/MM/AAAA HH:MM:SS`.

- **Paramètre** : `Date` — une date Zyllio
- **Retourne** : texte (ex: `25/02/2026 14:35:22`)
